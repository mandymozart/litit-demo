How it works

1. Choose a plan

Get started at here.

2. Add widget to page

Than copy and paste this widget code anywhere on your page.

<div data-litit="litit-demo.netlify.app"></div>
<script src="https://litit.netlify.app/cdn/litit.js"></script>

Make sure you replace the data-litit attribute with your DOMAIN_KEY.

If you want to use our presets, copy the stylesheet to your HTML header (optional): 

<link rel="stylesheet" href="https://litit.netlify.app/cdn/litit.css">

Custom Styling

We provide three CSS classes for styling.

  • .litit-title - the notes header
  • .litit-content - the content paragraph
  • .litit-updated - the last updates timestamp
